# 하루삼일

하루삼일은 24시간을 하나의 하루로 보지 않고, 8시간 단위의 3개 하루로 재해석하는 루틴 알람 서비스입니다.

## One-liner

하루를 3일처럼 사는 루틴 알람.

## Core Idea

보통 사람은 오전을 놓치면 "오늘은 망했다"고 느낍니다.
하루삼일은 그 감각을 바꿉니다.

> 첫 번째 하루는 끝났고, 두 번째 하루가 시작됩니다.

이 서비스의 목적은 완벽한 루틴을 강요하는 것이 아니라, 사용자가 하루 안에서 여러 번 다시 시작하게 돕는 것입니다.

## MVP Scope

- 24시간을 8시간씩 3개 하루로 분할
- 현재 몇 번째 하루인지 표시
- 각 하루마다 목표 1개 설정
- 하루 시작/종료 알림
- 완료, 놓침, 버팀 체크인
- 재시작 버튼
- 오늘의 3일 리포트
