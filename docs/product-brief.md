# Product Brief

## Product Name

하루삼일

## Positioning

하루삼일은 24시간을 3개의 하루로 나누어 8시간마다 새롭게 시작하게 해주는 루틴 알람입니다.

## Problem

많은 사용자는 아침 계획이 무너지면 하루 전체를 포기합니다.
기존 루틴 앱은 할 일, 체크리스트, 습관 달성률에 집중하지만 실패 이후 복귀 경험이 약합니다.

## Insight

사용자는 더 많은 목표가 필요한 것이 아니라, 망쳤다고 느끼는 순간 다시 시작할 수 있는 시간 구조가 필요합니다.

## Solution

하루를 8시간 단위의 3개 하루로 나눕니다.

- 1일차
- 2일차
- 3일차

각 하루에는 하나의 역할과 하나의 목표만 둡니다.
각 하루의 시작과 끝에서 알림과 체크인을 제공합니다.

## Target Users

### Primary

- 오전을 망치면 하루를 포기하는 사람
- 루틴 앱을 여러 번 깔았지만 지속하지 못한 사람
- 직장, 공부, 운동, 창작을 하루 안에 나눠 살고 싶은 사람

### Secondary

- 수험생
- 프리랜서
- 1인 사업자
- 사이드 프로젝트를 하는 직장인

## Value Proposition

하루삼일은 사용자가 하루를 포기하지 않게 만듭니다.
8시간마다 새 하루가 시작되므로, 오전을 놓쳐도 오늘은 아직 두 번 남아 있습니다.

## Core Message

- 하루는 하나가 아닙니다.
- 하루 한 번 사는 건 너무 낡았습니다.
- 오늘의 2일차가 시작됐습니다.
- 오늘은 아직 이틀 남았습니다.
- 8시간마다 새 하루가 열립니다.

## MVP Success Criteria

- 사용자가 첫 실행 1분 안에 3개의 하루를 생성한다.
- 홈 화면을 열었을 때 현재 일차와 남은 시간을 즉시 이해한다.
- 사용자가 하루에 최소 1번 체크인을 완료한다.
- 사용자가 "오전을 망쳐도 다시 시작할 수 있다"는 감각을 받는다.

## Monetization Hypothesis

Free:

- 3일 분할
- 기본 알림
- 일차별 목표 1개
- 7일 기록

Premium:

- AI 루틴 추천
- 주간 패턴 분석
- 위젯
- 알림 문구 커스터마이징
- 직장인/수험생/창업자 템플릿
- 무제한 기록
